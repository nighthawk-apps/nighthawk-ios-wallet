# Architecture

> **Note**: This file is kept for backward compatibility. The full architecture documentation is in [Darkfi_iOS_Architecture.md](Darkfi_iOS_Architecture.md).

Our App architecture is based on [The Composable Architecture](https://github.com/pointfreeco/swift-composable-architecture) by pointfreeco, integrated with the DarkFi P2P network via Rust UniFFI bindings.

See also:
- [DarkFi Integration Architecture](darkfi-integration.md) — layers, SDK, FFI, endpoint configuration
- [DarkIRC on iOS](darkirc-ios.md) — in-process darkirc via callback bridge
- [Feature Catalog](app-features.md) — iOS vs Android feature parity
